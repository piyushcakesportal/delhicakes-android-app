# DelhiCakes Flutter Android App

Connected prototype for https://delhicakes.com.

## Features
- Live WooCommerce products and prices
- Live product categories
- Product search
- Product detail pages
- Pull-to-refresh and pagination
- Existing DelhiCakes product page handoff for options, delivery slots and payment

The first version intentionally uses the existing WooCommerce product page for
weight, date, time slot, cake message, image upload and payment so custom plugins
continue to work while native checkout is developed.
